# Computer Vision BYOP: Cat vs. Dog Binary Classifier

## Project Overview
This project is a command-line interface (CLI) application designed to perform binary image classification (Cat vs. Dog). It utilizes a custom-trained ResNet-18 Artificial Neural Network model. The pipeline applies core Computer Vision concepts, including image normalization, tensor transformations, and pattern analysis. It is built to be fully executable from a headless terminal environment.
